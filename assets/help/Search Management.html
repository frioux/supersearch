Search Management

Search data is stored as three basic parts:
Name
The name that will be displayed in the spinner for the search.
URL
The URL that the search is based on; put %s in the URL to have the search insert whatever term you want in the URL.
Term
This is what you are actually doing when you search; for example you search on Google, but you Define on Merriam-Webster.
  1. From the main screen, press the menu button, and press the Manage Searches button.
  2. Update a search by selecting the search with the spinner, modifying any fields you would like to change, and pressing the Update Search button.
  3. Add a search by filling in data for each of the fields and then pressing the Add New Search button.
  4. Delete a search by selecting the search with the spinner and pressing the Delete Search button.
